# Convert2ASCII

Convert Image/Video to ASCII art.


## Prerequisites

* Ruby3+
* ImageMagick ([Download here](https://imagemagick.org/script/download.php))
* ffmpeg ([Download here](https://www.ffmpeg.org/))
